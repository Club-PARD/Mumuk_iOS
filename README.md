# 🍚 머먹_Mumuk

*💡서비스 한줄설명: 서로의 음식 선호도를 반영한 메뉴 추천 서비스💡*

디테일 설명: 얘들아! 오늘 ‘머먹’을래?

사람들과 함께 식사할 때, 메뉴 선정에 어려움을 겪어본 적 있나요?
’머먹‘을 통해 서로의 음식 취향을 공유하고, 공유된 취향을 반영한 메뉴 추천도 받아보세요!

얘들아! 오늘 이거 먹자!
